# Проект "Домашняя работа 10_1_"

## Описание:

модуль processing, masks, widget который содержит новые функции обработки данных.

## Установка:

1. Клонируйте репозиторий:
```
git clone https://github.com/username/myblog.git
```
2. Тестирование: скопируйте команду ниже, которая запускает 
тесты из папки src, также проверяет покрытие кода из папки src и генерирует HTML-отчет о покрытии
```
pytest --cov=src --cov-report=html  
```
